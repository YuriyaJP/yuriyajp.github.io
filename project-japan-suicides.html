Grants Operations Dashboard

A sheet-integrated dashboard demonstrating operational workflows used in grantmaking: data intake, record maintenance, compliance checks, communication readiness, and round tracking.

Home

Goal

Build a lightweight, transparent operations dashboard that integrates with a collaborative sheet (Google Sheets / Excel Online) to showcase the workflows teams use to intake applications, maintain records, run compliance checks, prepare communication packages, and manage grant rounds.

Tools Used

Python for website scraping, Google Sheets, Apps Script JS, HTML/CSS, lightweight JS for embedding, GitHub for code and automation scripts.

Key Features

  • Data intake: structured intake form that populates the sheet and creates a normalized applicant record.
  • Record maintenance: canonical records with status fields, ownership, and history notes.
  • Compliance checks: formula-driven checks and flags (attachments, eligibility, budget thresholds).
  • Communication readiness: templated email/letter generation columns and readiness indicators.
  • Round tracking: configurable rounds with roll-forward of eligible applications and summary KPIs.

Sheet Integration

The live dashboard is driven by a shared spreadsheet.
